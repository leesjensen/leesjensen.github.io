Transition

Transitions automatically associate specially named CSS selectors with an element based on v-if and v-show transitions.

I am exciting
 
<style>
  .fancy-leave-to {
    transform: scale(0);
    transition: transform 1s;
  }
  .fancy-enter {
    transform: rotate(0deg) scale(0);
  }

  .fancy-enter-to {
    transform: rotate(362deg) scale(1);
    transition: transform 1s;
  }
</style>

<div id="app">
  <button v-on:click="show = !show">Toggle</button>
  <transition name="fancy">
    <div v-if="show">I am exciting</div>
  </transition>
</div>
