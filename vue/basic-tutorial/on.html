On

v-on allows you to handle events. It may be abbreviated as @. Events can have modifiers such as the enter on keyup.

Currently: {{order}}

<div id="app">
  <p>Currently: {{order}}</p>
  <button v-on:click="set('pork')">Pork</button>
  <button @click="set('chicken')">Chicken</button>
  <input v-on:keyup.enter="onEnter" />
</div>

<script>
  var app = new Vue({
    el: '#app',
    data: { order: 'not set' },
    methods: {
      set: function (value) {
        this.order = value;
      },
      onEnter: function (e) {
        this.set(e.target.value);
      },
    },
  });
</script>
