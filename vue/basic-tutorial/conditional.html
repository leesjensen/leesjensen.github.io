Conditional

v-if, v-else actually remove elements from the DOM. v-show toggles the display state.

Current Number: {{clickCount}}

The number is: Zero Even Odd and below 10 

<div id="app">
  <p>Current Number: {{clickCount}}</p>
  <button v-on:click="++clickCount">Increment</button>
  <p>
    The number is:
    <span v-if="clickCount === 0">Zero</span>
    <span v-else-if="clickCount % 2 === 0">Even</span>
    <span v-else>Odd</span>
    <span v-show="clickCount < 10"> and below 10</span>
  </p>
</div>

<script>
  var app = new Vue({
    el: '#app',
    data: { clickCount: 0 },
  });
</script>
