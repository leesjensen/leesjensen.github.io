Attribute Binding

v-bind allows bind a element property to Vue instance data or any expression. It may be abbreviated as :. Binding the element's class property allows for the specification of multiple classes that are each applied based upon an expression.

Current State: {{state}} 

<div id="app">
  <p>
    Current State:
    <span v-bind:class="['trooper', {sel: state}]"
          v-bind:style="labelStyle">{{state}}</span>
  </p>
  <button v-bind:disabled="!state" v-on:click="toggle">false</button>
  <button :disabled="state" v-on:click="toggle">true</button>
</div>

<script>
  var app = new Vue({
    el: '#app',
    data: {
      state: true,
      labelStyle: {
        fontWeight: 'lighter',
      },
    },
    methods: {
      toggle: function () {
        this.state = !this.state;
        this.labelStyle.textTransform =
          this.state ? 'uppercase' : 'capitalize';
      },
    },
  });
</script>
