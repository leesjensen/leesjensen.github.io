Watcher

Watching a Vue instance data item will notify you when the data changes.

Path:

Security Log:

{{entry}}
<div id="app">
  <p>Path: <input v-model="input" /></p>
  <p>Security Log:</p>
  <div class="hlb" v-for="entry in securityLog">{{entry}}</div>
</div>

<script>
  var app = new Vue({
    el: '#app',
    data: {
      input: '',
      securityLog: [],
    },
    watch: {
      input: function (newInput, oldInput) {
        if (/(^\/|\.\.\/)/.test(newInput)) {
          this.securityLog.push(newInput);
        }
      },
    },
  });
</script>
