Methods

A Vue instance can contain methods and data. This makes it similar to a class.

{{format('Hey')}}
<div id="app">{{format('Hey')}}</div>

<script>
  var app = new Vue({
    el: '#app',
    data: { message: `i'm alive` },
    methods: {
      format: function (prefix) {
        return prefix + ' ' + this.message.toUpperCase();
      },
    },
  });
</script>
